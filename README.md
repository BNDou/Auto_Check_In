## 青龙部署

1. 拉库指令

国内机用下面指令（带代理），国外机把代理地址删掉即可：

```
ql repo https://ghproxy.com/https://github.com/BNDou/Auto_Check_In.git "checkIn_" "backUp" "^sendNotify"
```
